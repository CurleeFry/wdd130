Tropical Smoothie Cafe Labor Manager+

The purpose of this site is to help Managers and Shift Leaders better manage labor by breaking it up into different categories. It works using a python program that:
- Reads and stores current labor percentages
- Sorts labor by time and location
This website then accesses that information and displays it in where required. The weekly labor will be updated daily, in order to better show if the current week is relatively busier or slower.
